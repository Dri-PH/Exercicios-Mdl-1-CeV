Principais Formatações

Negrito / Destaque

A tag B é uma forma (não semântica) de se usar essa formatação.

A ta STRONG é uma (forma semântica) de se usar essa formatação.

Itálico / Ênfase

Usando a tag I temos uma forma não semântica de se usar a formatação em itálico.

Usando a tag EM temos uma forma semântica de se usar a formatação com ênfase.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto pequeno

Crinado um texto pequeno

Texto deletado

Pode-e maracar um texto como excluido para indicar que ele deve ser lido, mas n considerado

Texto inserido

Pode-se Marcar um texto como inserido para dar uma ênfase e indicar que foi adicionado depois.
